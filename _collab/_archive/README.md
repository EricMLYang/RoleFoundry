# _archive 歸檔區

已結案的 requests、responses、discussions 搬到這裡。

## 歸檔規則

1. request + 對應 response 一組搬入，保持配對關係
2. discussion 整個子資料夾搬入
3. 相關 artifact 留在 `artifacts/`（artifact 是結論，不歸檔）

## 目錄結構建議

```
_archive/
├── 2026-03/
│   ├── requests/
│   ├── responses/
│   └── discussions/
└── 2026-04/
    └── ...
```

按月份分組，避免單一資料夾過多檔案。
