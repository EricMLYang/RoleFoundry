# PM Repo Seed (Discovery 3-Stage)

此 seed 來自 MI_PM 的實戰流程，提供可直接複製的最小資料夾結構：

- `00_context/`：決策背景與約束
- `01_inbox/`：信號池
- `05_product/`：三階段核心流程（Outcomes -> OPP+Experiment -> Backlog/Handoff）

使用建議：
1. 先維持檔案結構不變，再逐步填入專案內容。
2. 三階段移動請搭配技能：
   - `pm-move-inbox-to-opp-exp`
   - `pm-move-opp-exp-to-backlog`
   - `pm-move-backlog-to-handoff`
