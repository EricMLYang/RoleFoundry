# OPP 區塊模板

插入 `05_product/02_opp_and_exp/0_OPP.md` 對應 Outcome `##` 分組下使用。

```md
### OPP-XXX [標題]
- 類型: Problem | Opportunity
- Level: Light | Medium | Deep
- Method: 商業/業務驗證 | User Evidence Test | Technical Spike | Data Quality Test | Operational Pilot
- 對齊 Outcome: OUT-XXX
- 狀態: identified | experimenting
- 信號來源: [SIG-ID]
- 核心問題：[一句話描述問題或機會]
- 驗證方式：[方法 + 對象 + 關鍵動作]
- 成功標準：[可量化門檻，建議用「至少 X% 的 Y 會 Z」]
- 下一步：[最小可行動作]
```
